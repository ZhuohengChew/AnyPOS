🌟 MSME Platform Guide

🚀 Getting Started

Welcome to MSME Finance

Your comprehensive financial management platform designed specifically for Malaysian SMEs.

Quick Start Guide

  1. Language Selection: Choose your preferred language (EN/MY/ZH/TA)
  2. Account Type: Select Personal or Business user
  3. Business Setup: Enter your business details and LHDN registration
  4. Bank Connection: Link your Malaysian bank accounts
  5. Start Managing: Access dashboard and begin using features

Dashboard Overview

  • Quick Actions: Pay, Receive, Invoices, Analytics
  • Account Balances: Real-time bank account overview
  • Recent Transactions: Latest payment activities
  • ESG Metrics: Sustainability tracking
💡 Tip: Use the tutorial button (📖) anytime to access this guide.

💳 Payment Features

Making Payments

  1. Click "Pay" from dashboard
  2. Enter recipient details (supplier/vendor)
  3. Select payment method (bank transfer/QR)
  4. Enter amount and add description
  5. Confirm with PIN/biometric

Receiving Payments

  1. Click "Receive" from dashboard
  2. Generate payment QR code
  3. Share with customer
  4. Track payment status
  5. Auto-categorize transactions

Security Features

  • PIN/biometric authentication
  • Transaction limits
  • Fraud detection
  • Secure bank connections
🔒 Security: All transactions are encrypted and comply with Malaysian banking standards.

🧾 Invoice Management

Creating LHDN-Compliant Invoices

  1. Click "Invoices" from dashboard
  2. Enter customer information
  3. Add line items with descriptions
  4. Automatic SST (6%) calculation
  5. Preview and send invoice

Malaysian Compliance

  • LHDN-compliant format
  • Automatic SST calculation
  • Tax identification numbers
  • Digital signature support

Invoice Features

  • Save as draft
  • Email integration
  • Payment tracking
  • Template customization
📋 Compliance: All invoices automatically include required LHDN fields.
